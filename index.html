  • D. Passe chaque élément au serveur

  • A. Oui
  • B. Non
  • C. Rien
  • D. Peut-être

3. Comment afficher les nombres de 1 à 5 en PHP en utilisant une boucle for ?

  • B. for ($i = 5; $i > 0; $i--) { echo $i; }

  • D. Demande au serveur de compter

4. Quelle est la bonne syntaxe pour une condition PHP ?

  • A. if ($x > 0) { ... }
  • B. if x > 0 { ... }
  • C. if x > 0: ...
