Python

1. Comment afficher "Hello, World!" en Python, tout en utilisant une méthode qui répète chaque lettre deux fois ?

  • A. print('HHeelllloo,, WWoorrlldd!!')
  • B. print('Hello, World!' * 2)
  • C. print(''.join([char * 2 for char in 'Hello, World!']))
  • D. Répète chaque lettre à haute voix pendant que tu écris

2. Comment déclare-t-on une fonction Python qui retourne la somme des carrés des éléments d'une liste donnée ?

  • A. def somme_carres(liste): return sum([x**2 for x in liste])
  • B. def somme_carres(liste): return sum(liste) ** 2
  • C. def somme_carres(liste): print(sum([x * 2 for x in liste]))
  • D. Demande à un mathématicien d'écrire la formule pour toi

3. Comment créer une liste en Python contenant les carrés des nombres de 1 à 10, mais uniquement pour ceux divisibles par 3 ?

  • A. ma_liste = [x**2 for x in range(3, 10)]
  • B. ma_liste = [x**2 for x in range(1, 11) if x % 3 == 0]
  • C. ma_liste = list(range(1, 11, 3))
  • D. Fais un sondage pour connaître les préférences de Python

4. Quelle est la bonne manière d’importer toutes les fonctions d’un module Python tout en attribuant un alias au module ?

  • A. import module as alias
  • B. from module import *
  • C. from module import * as alias
  • D. Télécharge le module, grave-le sur un CD et renomme le CD

JavaScript

1. Comment afficher "Hello, World!" en JavaScript, mais en inversant les caractères de la chaîne avant de l’afficher ?

  • A. console.log([...‘Hello, World!’].reverse().join(''))
  • B. console.log('Hello, World!'.split('').reverse().join(''))
  • C. alert('Hello, World!') // inversé automatiquement
  • D. Appelle un magicien qui renverse les chaînes pour toi

2. Comment déclarer une constante en JavaScript tout en s’assurant qu’elle ne puisse pas être modifiée ?

  • A. let PI = Object.freeze(3.14);
  • B. const PI = 3.14;
  • C. var PI = 3.14;
  • D. Gravée sur une pierre antique, immuable pour toujours

3. Comment définir une fonction fléchée en JavaScript pour doubler chaque élément d’un tableau donné ?

  • A. function doubler(arr) { return arr.map(x => x * 2); }
  • B. const doubler = (arr) => arr.map(x => x * 2);
  • C. fun doubler(arr) => arr.map(x -> x * 2);
  • D. Crée un miroir magique pour doubler les éléments

4. Quelle est la principale différence entre Object.freeze() et Object.seal() en JavaScript ?

  • A. seal bloque tout, freeze ne fait rien
  • B. freeze empêche tout changement, seal empêche l’ajout/suppression de propriétés
  • C. Les deux font exactement la même chose
  • D. freeze fait geler les objets littéralement, seal les enferme dans un coffre

PHP

1. Comment afficher "Hello, World!" en PHP avec une balise HTML h1 ?

  • A. <?php echo '<h1>Hello, World!</h1>'; ?>
  • B. <?php print '<h1>Hello, World!</h1>'; ?>
  • C. <?php echo h1('Hello, World!'); ?>
  • D. Demande au navigateur de l'afficher directement

2. Comment déclarer une constante en PHP qui ne peut pas être redéfinie ?

  • A. define('PI', 3.14);
  • B. const PI = 3.14;
  • C. $PI = 3.14;
  • D. Écris-la sur un parchemin sacré !

3. Comment définir une fonction en PHP qui retourne la somme des éléments d’un tableau ?

  • A. function somme($array) { return sum($array); }
  • B. function somme($array) { return array_sum($array); }
  • C. def somme($array) { return sum($array); }
  • D. Fais appel à un comptable pour additionner !

4. Quelle est la différence entre un tableau indexé et un tableau associatif en PHP ?

  • A. Les tableaux indexés utilisent des chaînes comme clés, les tableaux associatifs utilisent des entiers
  • B. Les tableaux indexés utilisent des clés numériques, les tableaux associatifs utilisent des clés personnalisées
  • C. Il n'y a pas de différence entre les deux
  • D. Les tableaux indexés sont plus rapides car ils ont moins de personnalité
